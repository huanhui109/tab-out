# Tab Out

**掌控你的标签页。**

Tab Out 是一个 Chrome 扩展程序，它用一个仪表板取代你的新标签页，展示你当前打开的所有标签页。标签页按域名分组，主页（Gmail、X、LinkedIn 等）会被归入单独的分组。关闭标签页时还有流畅的音效和五彩纸屑动画。

无需服务器、无需账号、无需外部 API 调用。仅是一个 Chrome 扩展程序。

## 功能特性

- **一眼看遍所有标签页** - 简洁的网格布局，按域名分组显示
- **主页分组** - 将 Gmail 收件箱、X 首页、YouTube、LinkedIn、GitHub 主页合并到一个卡片中
- **优雅关闭标签页** - 关闭时伴有音效和五彩纸屑爆发动画
- **重复标签页检测** - 当同一页面打开多次时标记提醒，支持一键清理
- **点击跳转** - 点击任意标签页标题即可跳转到该标签页，不会打开新标签页
- **稍后阅读** - 将标签页保存到待办清单，稍后再处理
- **本地服务器分组** - 在标签页旁显示端口号，便于区分不同的开发项目
- **可展开的分组** - 默认显示 8 个标签页，点击"+N 更多"展开查看全部
- **100% 本地运行** - 你的数据永远不会离开你的设备
- **纯粹的 Chrome 扩展** - 无需服务器、无需 Node.js、无需 npm，只需加载扩展程序即可

## 背景模式

Tab Out 支持三种背景模式：

1. **无（默认）** - 简洁的纯色背景
2. **必应壁纸** - 每日自动从必应获取壁纸（点击箭头浏览最近 8 张图片）
3. **自定义图片** - 从本地文件上传自己的图片

扩展程序会自动从背景图片中提取主色调，并应用微妙的遮罩层，确保文字清晰可读。

## 侧边栏模式

Tab Out 可以作为侧边栏在任何网页上使用：

1. 右键点击 Chrome 中的 Tab Out 扩展程序图标
2. 选择"打开侧边栏"
3. Tab Out 将作为窄边栏出现在当前页面的右侧

功能特点：
- 手动刷新按钮更新标签页列表
- 创建新标签页或页面加载完成时自动刷新
- 针对窄边栏宽度优化的布局

## 手动安装

**1. 克隆仓库**

```bash
git clone https://github.com/zarazhangrui/tab-out.git
```

**2. 加载 Chrome 扩展程序**

1. 打开 Chrome 并访问 `chrome://extensions`
2. 启用**开发者模式**（右上角开关）
3. 点击**加载已解压的扩展程序**
4. 选择克隆仓库中的 `extension/` 文件夹

**3. 打开新标签页**

你会看到 Tab Out 界面。

## 工作原理

```
你打开一个新标签页
  -> Tab Out 按域名分组显示你打开的标签页
  -> 主页（Gmail、X 等）会在顶部单独分组
  -> 点击任意标签页标题即可跳转过去
  -> 关闭不需要的分组（伴随音效和五彩纸屑）
  -> 关闭前可以将标签页保存到稍后阅读
```

所有功能都运行在 Chrome 扩展程序内部。无外部服务器、无 API 调用、数据不会发送到任何地方。保存的标签页存储在 `chrome.storage.local` 中。
